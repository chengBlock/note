# 数据结构和算法的重要性

- 算法是程序的灵魂
- 大厂面试必考点

- 程序 = 数据结构 + 算法

**数据结构**是一门研究组织数据方式的学科,学好数据结构可以编写出更加漂亮的代码,更加有效率的代码.但并不意味着学好数据结构就一定能学好算法.

数据结构是算法的基础,要想学好算法,需要将数据结构学到位.

数据结构:

- 线性结构
  - 数据元素之间存在一对一的线性关系
  - 顺序表、链表
  - 数组、队列、链表、栈
- 非线性结构
  - 二位数组、多维数组、广义表、树、图
